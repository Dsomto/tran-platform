# Capstone resources — stage data files

Each subfolder contains the files referenced in that stage's mission brief.

**To deploy:**

1. Open `stage-0/`. Upload every file inside it to a new Google Drive folder.
2. Right-click the folder → **Share** → **Anyone with the link** → **Viewer**.
3. Copy the share link.
4. Repeat for `stage-1/` through `stage-4/`.
5. Paste the five share links into your next message and I'll wire them into the platform.

These files live outside `src/` and `public/` on purpose — they should not be bundled with the deployed app. Drive is the canonical home.
